Настройки

Внимание: Для выполнения действий на странице "Настройки" Вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у Вас роль, перейдите на страницу "Роли участников". Основные сведения о ролях и правах доступа изложены также в "Права доступа и роли: обзор".

Страница "Настройки" представляет собой страницу для установки по умолчанию настроек инструмента "Отправка", которые используются на сайте. Для эффективного использования инструмента "Отправка" необязательно вносить в Настройки какие-либо изменения. Тем не менее, страница позволяет гибко менять тип просмотра получателей, редакторы текста письма и другие функции.

Первой и самой существенной настройкой является выбор типа получателей. Выпадающее меню предлагает четыре варианта:

  • Пользователи - простой список всех пользователей в алфавитном порядке; особенно подходит для сайтов проекта или других небольших типов сайта, где отображение ролей участников сайта при использовании инструмента "Отправка" не является обязательным.
  • Пользователи по Роли - тип просмотра выбора пользователей по умолчанию; сортирует пользователей согласно приписанной им роли. Могут быть выбраны все участники сайта, обладающие данной ролью, или индивидуальные получатели. При этом типе просмотра выбора можно выбирать группы или блоки только среди участников с ролью "Студенты".
  • Параллельные списки - представляет собой два параллельно расположенных окна со списками; аналогичен типу просмотра списков Членства в группе. Участники сайта располагаются в списке согласно их ролям и могут быть выборочно добавлены к списку получателей. При этом типе просмотра выбора можно также выбирать группы и блоки, причем не только среди участников с ролью "Студенты".
  • Список прокрутки - представляет собой список, снабженный линейкой прокрутки, позволяющий выбирать участников сайта, удерживая клавиши "Ctrl" и "Shift".

Второй набор настроек предназначен для установки режимов копирования писем. Два возможных варианта - "Отправить мне копию" и "Добавить в "Архив электронной почты"" также доступны на странице "Написать письмо", однако, установленые на странице "Настройки", они являются установками по умолчанию для всех участников сайта, использующих инструмент "Отправка" на этом сайте.

Третий набор настроек определяет режимы ответов на отправленной пользователем письмо. Ответы направляются на электронный адрес отправителя. Можно также выбрать вариант настройки, запрещающий ответ на письмо, в таком случае обратный адрес просто меняется на 'no-reply@server.edu'. Настройки, выбранные здесь становятся настройками по умолчанию для всех участников сайта, использующих инструмент "Отправка" на этом сайте.

Четвертый набор настроек касается непостредственно текста письма, для создания которого может использоваться редактор в режиме точного отображения (HTML), установленный в файле формата sakai.properties; или простой редактор для неформатированных текстов.

И последнее. Если файл sakai.properties позволяет, можно изменить единственное и множественное число для ролей. Это позволит преподавателям присоединять их собственные имена к ролям участников сайта (по крайней мере, при использовании инструмента "Отправка") и изменять по своему усмотрению названия ролей в типе просмотра выбора, если ролям не соответствует правильное грамматическое число.

после нажатия кнопки "Обновить" настройки инструмента "Отправка" сохраняться по умолчанию для всего сайта. После нажатия кнопки "Отмена" Вы вернетесь на страницу "Написать письмо", настройки по умолчанию сохранены не будут.
