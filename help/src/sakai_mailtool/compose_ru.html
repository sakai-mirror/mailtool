Как выбрать получателей

Инструмент "Отправка" содержит четыре различных типа выбора пользователей (дополнительные сведения о типах выбора пользователей смотрите   Настройки, Сохранить по умолчанию. По умолчанию задан тип "Пользователи по роли".

Сверху страницы "Написать письмо" в поле "От:" будет отображаться электронный адрес текущего пользователя. Должен ли этот адрес автоматически появляться в поле "Кому:", после того, как получатель Вашего письма нажмет кнопку "ответить", необходимо задать на странице Настройки (смотрите Настройки, Сохранить по умолчанию ).

В разделе "Кому:" осуществляется выбор получателей, необходимо поставить флажок, чтобы обозначить желаемых получателей. Выбор касатся ролей, блоков, групп, индивидуальных получателей, или любых комбинаций вышеперечисленных форм организации участников.

В первую очередь списки участников формируются по роли. Занести в список можно желаемых участников, обладающих разными ролями. Для этого необходимо нажать ссылку "Выбрать <название роли>". Перед Вами появится полный список участников, обладающих выбранной ролью. Название ссылки изменится на "Свернуть" для свертывания подробного списка участника. Получатели перечислены в алфавитном порядке пофамильно. (В инструменте Отправка выбор осуществляется подобным образом для каждой роли.)

После раздела "Кому:" следует поле "Другие получатели", в которое можно внести дополнительные электронные адреса. Обычно это поле используется для адресов получателей, неявляющихся участниками сайта.

Тема и тело письма

В строке "Тема:" по умолчанию отображается название сайта. При желании тему письма можно изменить. Для использования доступны два текстовых редактора: для создания неформатированного текста и редактор для создания сообщений в режиме точного отображения (HTML). Редактор, заявленный в файле sakai.properties, используется по умолчанию. Однако, можно выбрать дургой редактор в Настройках.

Как приложить файлы

Прямо под разделом для выбора получателей расположена ссылка, позволяющая пользователям приложить файл к письму. После нажатия этой ссылки появляется поле с кнопкой "Обзор": пользователь может найти необходимый файл в своем компьютере. После выбора прилагаемого файла нужно кнопки "Открыть" и "Приложить". Обратите внимание, что приложениями являются файлы, добавляемые только из компьютера пользователя; таким образом, работа с ними отличается от действий с ресурсами или другими приложениями Sakai.

Хранение писем

Под окном с текстом письма можно выбрать варианты хранения письма. Первый - "Отправить мне копию" позволяет отправителю сохранить копии его собственных исходящих сообщений. Выбор второго - "Добавить в "Архив электронной почты" - предполагает, что копия письма с помощью инструмента "Архив электронной почты". Обратите внимание, что выбор этого варианта предполагает возможность просмотра письма всеми участниками. Режим по умолчанию для обоих вариантов может быть установлен пользователем, обладающим статусом "maintain"(смотрите Настройки).

Отправка письма и страница подтверждения

Отправка письма осуществляется нажатием кнопки "Отправить письмо"; письмо будет доставлено всем предварительно выбранным получателям. После отправки появляется страница подтверждения, где под заголовком "Письмо отправлено:" отображается список выбранных получателей письма.
