BU Mailtool Setup Instructions for Sakai 2.1.2

Requirements

  • Sakai 2.1.x

Features

  • Basically original Mailtool + Wysiwyg HTML editors
  • Support two popular Wysiwyg editors: HTMLArea and FCKeditor
  • File linking in HTMLArea (Insert Image/Link buttons)
  • Messages are encoded in HTML (text/html)

Build & Installation (Windows XP)

  1. Edit build.properties (Change the location of tomcat & repositories if needed)
  2. Stop Tomcat.
  3. Run "maven" in the top of the source directories
  4. IF "BUILD SUCESSFUL", Now it's ready.
  5. If "BUILD FAIL", check error messages and location of java compiler/library/repositories.
  6. Restart tomcat (Sakai Server).
  7. (If you want to add Mailtool as a new tool)
  • Go to My Workspace->Worksite setup->Check a box which you want to add the tool->Click "Revise"->Click "Edit Tools"
  • ->Check Mailtool->Click "Continue" then "Finish". (See "Configuration" below)
  1. (If you already added Mailtool) do nothing. (See "Configuration" below)

Configuration

  • Set editor-type (You should be able to maintain the site)
  • My Workspace->Sites->Click a site id-> Click "Pages" box-> Click the site id of Mailtool->Click tool ID
  • Fill in "wysiwygeditor" with "FCKeditor" or "HTMLArea" (case-insensitive). Other string will be ignored(default: plain text editor)
  • For general configuration of mailtool
  • http://bug.sakaiproject.org/confluence/display/MAILTOOL/Home
  • http://bug.sakaiproject.org/confluence/pages/viewpage.action?pageId=10181
