Mailtool: Overview

Mailtool allows user to send a message to the site particpants. Also it provides wysiwyg editing, file attachment.

Compose

On the Compose page, you can select recipient(s) from the pre-configured recipient view. If you want to send a copy to non-site-participant, enter the recipent's email address in the "Other recipient(s)" text field.

Options

On the Options page, you can set the defaults: recipient view, send me a copy, Add to Email Archive, reply-to, message format. Also, only when sakai.properties has "mailtool.show.renaming.roles=true", the allowed role can rename the role names.
