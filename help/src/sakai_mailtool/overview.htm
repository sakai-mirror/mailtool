Mailtool: Overview

Mailtool allows user to send a message to the site particpants. Also it provides wysiwyg editing, file attachment, group/section-aware recipient selection.

Compose

On the Compose page, you can select recipient(s) from the pre-configured recipient view. If you want to send a copy to non-site-participant, enter the recipent's email address in the "Other recipient(s)" text field. If you want to attach file(s), click "Attach a file". Enter the subject and message. If Wysiwyg editor is configured for text editing, you should be able to edit message in HTML format. Finally check "Send me a copy" and "Append to Email Archive" if you want to.
Note: Attached file will not be archived in Email Archive. Only attachment info (file name and size) will be recorded.

Options

On the Options page, you can set the defaults: recipient view, send me a copy, Add to Email Archive, reply-to, message format. Also, only when sakai.properties has "mailtool.show.renaming.roles=true", the allowed role can rename the role names.
