Browse/Insert a link in your Site's Resouces

This form uses a customised HTMLArea. When in edit mode you should see a button with this icon . This icon lets you link to other documents in this database.

  • Type & select/Select some text in the editor and press the icon
  • Choose a document from the list and click on it.
  • Press Ok
  • Press the "Save Document" button at the bottom of the form

If all goes well it will save a document that contains a link to another document that uses this same form. You can then open another document, edit it, link to another document ad infinitum, Great for intranets!!
